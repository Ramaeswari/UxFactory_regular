Html

Firstly we learned about basic html code. Html5 starts with doctype tag, this tells to browser as this is webpage with html code. Later we learned about html tags and how to create lists, links,images, tables, forms so on. Some imprtant tags are:

  • For creating lists we use ol, ul, dl tags. Inside ul and ol tags we have li tag
    1. ol tag: is used to create ordered list with ol tag. This list displays with numbers.
    2. ul tag: is used to create unordered list with ul tag. This list displays with bullets.
    3. dl tag: is used to create discription list with dl tag. Inside we have dt and dd tags. dt tag means term what we want to discripts and dd tag means it describes the term what we write dt tag.
  • For creating links we use anchor tag i.e a tag. Inside a tag we have href attribute, inside this attribute we give source of our link.
  • For creating images in our webpages we use img tag. Inside img tag we have src attribute, inside this attribute we give source of our image.
  • For creating tables we use table tag, inside this we have tr tag for rows and td tag for table columns.

CSS

Css is used to style our webpage. In css we have several properties for styling html elements.

  • We learned selectors , in css we have several selectors. They are simple selectors, combination selectors, pseudo class selectors, pseudo element selectors, attribute selectors.
